Read data point(s)

通过id和过滤条件去查询对应客户端下的数据点。

REQUEST

GET

/client/{clientId}
Parameters
clientId string c24b0fe51856497eebb6a2bfcd120247aac0d6334d670bb92e09a00ce8169365

客户端的唯一标识符，使用sdk进行生成。
room string livingroom

查询与某个房间相关的数据点。
device string light

查询与某个电子设备相关的数据点。
limit number 10

查询的结果数。
start string 2018-10-10 09:30:00

起始时间，查询时间段内的数据。
end string 2018-10-10 10:30:00

结束时间，查询时间段内的数据。
min number 10

查询数据的最小值。
max number 50

查询数据的最大值。
sum number 30

查询指定分钟内的数据总值。
avr number 30

查询指定分钟内的数据平均值。
med number 30

查询指定分钟内的数据中位数。
cb string func

JSONP跨域请求的函数名。

Write data point(s)

往房间内的电子设备节点提交数据，创建时间由服务器生成。

REQUEST

POST

/client/{clientId}
Parameters
clientId string c24b0fe51856497eebb6a2bfcd120247aac0d6334d670bb92e09a00ce8169365

客户端的唯一标识符，使用sdk进行生成。
room string livingroom

提交的数据点所在的房间。
device string light

提交的数据点对应的电子设备。
data double 1.00

提交的数据点。

Authorization {JWT}
